## ✅ Project Description
This project simulates guided hiking tours using object-oriented programming principles. Participants and their luggage are redistributed using a greedy algorithm depending on the tour type (Family or Extreme).

## 🧩 Features
- Object-oriented design with abstract classes and interfaces
- Inheritance via overloading and overriding
- Markdown output with linked tour reports
- Console output matching expected distribution
- JUnit tests for the greedy algorithm logic
- CI/CD pipeline setup with `.gitignore` and no IDE files pushed

## 🔧 Technologies Used
- Java 17 (Eclipse Temurin)
- JUnit 5
- GitLab CI/CD
- Markdown for file output
